# Trustworthy Orchestration Criteria

## Enforced Criteria

- C1 Human Governance: blocks unauthorized autonomous mutations and requires human approval.
- C2 Policy-Enforced Operation: evaluates machine-readable policy rules for regulated actions.
- C6 Epistemic Prudence: escalates low-confidence recommendations.
- C7 Incremental Knowledge Evolution: records versioned, reversible knowledge updates.

The machine-readable report is exposed at `/api/v1/compliance/trustworthy-orchestration`.
